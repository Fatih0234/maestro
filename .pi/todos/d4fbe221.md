# Phase 2 Config Setup: OpenCode Profile Integration

## Objective

Implement Option B: Make OpenCode profile path configurable in WORKFLOW.md, allowing different OpenCode profiles (ws, omo-power, etc.) to be selected per workspace/issue.

## Background

We have verified through experiments that:

1. **OPENCODE_CONFIG env var works** - `OPENCODE_CONFIG=~/.config/opencode/profiles/ws/opencode.jsonc opencode serve` successfully loads the ws profile with all custom agents, skills, and MCP integrations.

2. **Agent selection works** - Sending `{"agent": "scribe", "parts": [...]}` in the message body selects the correct agent.

3. **Our ws profile contains** - `minimax-coding-plan/MiniMax-M2.7` model, custom agents (build, plan, scribe, coder, explore, etc.), MCP integrations (context7, exa, gh_grep), skills and commands.

4. **Model is set at startup** - Runtime PATCH /config does not reliably change the model. Model must be set via OPENCODE_CONFIG at startup.

**C. In `startServer()` method, inject environment variables:**
```go
func (r *OpenCodeRunner) startServer(...) {
    // ... existing code ...
    
    cmd := exec.Command(argv[0], argv[1:]...)
    cmd.Dir = workDir
    
    // NEW: Inject OPENCODE_CONFIG pointing to profile
    profilePath := os.ExpandEnv("$HOME/.config/opencode/profiles/") + r.profile + "/opencode.jsonc"
    cmd.Env = append(cmd.Env, "OPENCODE_CONFIG="+profilePath)
    
    // NEW: Optionally inject OPENCODE_CONFIG_DIR
    if r.configDir != "" {
        cmd.Env = append(cmd.Env, "OPENCODE_CONFIG_DIR="+r.configDir)
    }
    
    // Inherit existing environment (PATH, auth keys, etc.)
    for _, e := range os.Environ() {
        if !strings.HasPrefix(e, "OPENCODE_") {
            cmd.Env = append(cmd.Env, e)
        }
    }
}
```

## Key Technical Details

### Environment Variable Injection

When calling `exec.Command`, we must:
1. Inherit the current process environment (for PATH, API keys, etc.)
2. Add our OPENCODE_CONFIG and optionally OPENCODE_CONFIG_DIR
3. Not overwrite existing OPENCODE_* vars unnecessarily

```go
cmd.Env = os.Environ() // Start with current environment
cmd.Env = append(cmd.Env, "OPENCODE_CONFIG="+profilePath)
if configDir != "" {
    cmd.Env = append(cmd.Env, "OPENCODE_CONFIG_DIR="+configDir)
}
```

### Agent Selection in Message Body

The message body accepts `agent` as a string field:
```json
{
  "agent": "scribe",
  "parts": [{"type": "text", "text": "..."}]
}
```

### Profile Path Resolution

Profile name → absolute path:
```
ws → $HOME/.config/opencode/profiles/ws/opencode.jsonc
omo-power → $HOME/.config/opencode/profiles/omo-power/opencode.jsonc
```

Use `os.ExpandEnv("$HOME/...")` to resolve home directory.
